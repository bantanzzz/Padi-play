      🎮 How to Play

      1. Set your profile name and emoji
      2. Create a room or join with a room code
      3. Host starts a round - everyone gets the same letter!
      4. Fill in all 5 categories before time runs out:
      • Name: Person's name
      • Place: City, country, or location
      • Animal: Any animal name
      • Thing: Object or item
      • Artist: Singer or musician
      5. Faster submissions = more points! 🏆

      ⚡ Scoring & Tips

      Speed Bonus System
      Submit faster = more points! Points equal remaining time when you submit.
